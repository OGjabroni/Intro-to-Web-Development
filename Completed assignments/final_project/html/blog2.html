Deleting Contacts from Salesforce Marketing Cloud

Salesforce Marketing Cloud is a powerful platform that enables businesses to manage their marketing campaigns and engage with customers effectively. However, organizations may often find the need to remove certain contacts from their database for various reasons. In this article, we'll explore how to efficiently delete contacts from Salesforce Marketing Cloud using Script Activity and Server-Side JavaScript (SSJS).

Understanding Script Activity

Script Activity allows users to execute custom codes within Salesforce Marketing Cloud to carry out specific tasks. It is a versatile tool that integrates SSJS, AMPscript, and other scripting languages. By utilizing Script Activity, we can automate the process of deleting contacts from the database. You can achieve this by following the steps below:

  1. Accessing Automation Studio

    To begin, access Automation Studio in Salesforce Marketing Cloud. Automation Studio is an application that provides a workflow engine to automate and manage various marketing processes. Here, we can create and schedule activities, including Script Activities.

  2. Creating a Script Activity

    Within Automation Studio, create a new Script Activity by selecting "Create Activity" and then choosing "Script Activity." Provide a suitable name and description for the activity to easily identify it later.

  3. Writing the SSJS Code

    In the Script Activity configuration, navigate to the "Code" tab. Here, you can write the SSJS code needed to delete contacts. Below is an example code snippet:

    <script runat="server"> Platform.Load("Core","1"); var contactsToDelete = ["contactKey1", "contactKey2", "contactKey3"]; var apiEndpoint = "https://YOUR_SUBDOMAIN.rest.marketingcloudapis.com/address/v1/contacts/delete"; for(var i=0; i<contactsToDelete.length; i++){ var restResponse = HTTP.Delete(apiEndpoint + "?key=" + contactsToDelete[i]); Write(restResponse.Content); } </script>

    In the code snippet, contactsToDelete represents an array containing the contact keys of the contacts to be deleted. Replace the sample contact keys with the actual ones you wish to remove from the Salesforce Marketing Cloud. apiEndpoint defines the REST API endpoint used for performing delete operations. Ensure you replace "YOUR_SUBDOMAIN" with your Marketing Cloud subdomain.

    The for loop cycles through each contact key within the contactsToDelete array and sends a DELETE request to the API endpoint. The response is then written to the console.

  4. Executing the Script Activity

    Once you've written the code, save and run the Script Activity. It will initiate the process of deleting the specified contacts from the Salesforce Marketing Cloud database

Conclusion

By utilizing Script Activity and SSJS, businesses can efficiently delete contacts from Salesforce Marketing Cloud. The customizable nature of Script Activity allows for automating various processes, saving time and effort. Remember to exercise caution while deleting contacts, as the action cannot be undone.
